# 项目

```
思路：

markdown文件夹  =>（compile编译）   =>  data.json串
                                   =>  生成菜单目录            =>   build        =>  github.io可用
                                   =>  搜索关联 
```



# 暴露命令

```

npm run compile  编译

npm run build    打包

npm run dev      调试

```



# 过程说明

```
  compile过程处理markdown文件转换问题，
  输出[菜单的基础结构、搜索的json串]


  build过程主要是基于React的前端构建
```
